How To Implement



Place this html code into your html file where you want to show the animation.

<div class="carousel-container" id="carousel-1"> <div class="carousel-image-wrapper"> <img src="./images/1.jpg" alt="Image" /> <img src="./images/2.jpg" alt="Image" /> <img src="./images/3.jpg" alt="Image" /> </div> </div> </div>

Add your own id of carausel. i.e carousel-1 in example

add all images that you need to show at the time of slide

place this code in your javascript

const x = new Carousel({ carouselId: "carousel-1", transitionSpeed: 500, holdInterval: 2000, autoSlide: false, });

Provide carauselId, transitionSpeed (if you want to controll speed by your own), holdInterval(if you want to put your own),autoSlide:false (if you want to stop auto slide, which is optional, default value is true)

this initialize you carouesl by in this way and that's it.
